# typescript-backend-best-practice Repo

## Setup

- Clone repo
- npm install
- npm run dev to run dev server with ts-node
- npm run postgres setup docker container with postgres
- npm run createdb createdb with required name
- npm run migrateup **TODO**

## Features

1. Custom logging using morgan, Influence by Gin Go framework
2. Custom Error handling
3. Create migration by using npm run create-migration -- migration-name
